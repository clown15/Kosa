#### 문자열 포매팅
문자열의 format 메서드 사용하는 방법

    msg = ‘{} test text{}’
    msg.format(‘1,2’) -> 1 test text 2

    순서 지정
        msg = ‘{0}{1}{0}
        msg.format(‘1’,’2’) -> 121
    인자 이름 지정
        msg = ‘{one} {two} {one}’
        msg.format(one=’1’,two=’1’) -> 1 2 1 = msg.format(two=’2’,one=’1’)

f-string
문자열 앞에 f를 붙여 사용한다
출력할 내용 바인딩 필요
>a = ‘1’   
b = ‘2’   
msg = f’{a} {b} {a}’ -> 1 2 1
